## Descripción general

Este repositorio contiene el portafolio de proyectos desarrollados durante el semestre en la materia de Desarrollo de Aplicaciones Web. En este portafolio se integran diferentes prácticas y proyectos relacionados con el desarrollo de aplicaciones web, el uso de tecnologías del lado del cliente y del servidor, la creación de sistemas funcionales y el consumo de servicios mediante API.

El objetivo principal es demostrar la organización de los proyectos, el uso de GitHub, la documentación técnica, las evidencias de desarrollo y los aprendizajes adquiridos durante el semestre.

## Competencias desarrolladas

- Organización de proyectos web.
- Uso de Git y GitHub.
- Documentación técnica de proyectos.
- Desarrollo de aplicaciones web.
- Manejo de código fuente.
- Uso de tecnologías como HTML, CSS, PHP, WordPress y API REST.
- Integración de evidencias y capturas de pantalla.
- Reflexión sobre el aprendizaje obtenido.

## Proyectos incluidos

### Proyecto 01 — Mini-aplicación web

Este proyecto corresponde a la actividad de evaluación 1.2, donde se desarrolló una mini-aplicación web aplicando conceptos básicos de desarrollo web.

### Proyecto 02 — Gestor de tareas

En este proyecto se desarrolló una aplicación para gestionar tareas, permitiendo organizar actividades mediante una interfaz web.

### Proyecto 03 — Sistema de citas

Este proyecto consistió en el desarrollo de un sistema de citas, aplicando conceptos de programación web para registrar, organizar y administrar información.

### Proyecto 04 — Portafolio en WordPress

En este proyecto se creó un portafolio utilizando WordPress, con el objetivo de presentar información personal, académica o profesional mediante un sitio web administrable.

### Proyecto 05 — REST API Dog API

Este proyecto integrador consistió en consumir una API REST relacionada con razas de perros, aplicando peticiones web y manejo de información obtenida desde un servicio externo.
