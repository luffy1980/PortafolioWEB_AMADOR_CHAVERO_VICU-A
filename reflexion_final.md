# Reflexión final

## ¿Cuál proyecto fue más difícil?

Los proyectos que presentaron mayor dificultad fueron el Proyecto 02 — Gestor de tareas y el Proyecto 04 — Portafolio WordPress. El Proyecto 02 fue difícil porque el alumno tuvo que trabajar con operaciones como registrar, editar, eliminar y consultar información, además de organizar correctamente la conexión con la base de datos y los archivos del sistema. 

El Proyecto 04 también presentó dificultad porque se trabajó con WordPress, su estructura de carpetas, la personalización del sitio y la organización del contenido visual. Aunque los demás proyectos fueron más fáciles de realizar, estos dos requirieron más tiempo, revisión y pruebas para lograr un buen resultado.

## ¿Cuál proyecto fue más útil?

Los proyectos más útiles fueron el Proyecto 02 — Gestor de tareas y el Proyecto 04 — Portafolio WordPress. El Proyecto 02 fue útil porque permitió practicar funciones importantes de una aplicación web real, como crear, editar, eliminar y consultar datos. El Proyecto 04 fue útil porque ayudó a comprender cómo se puede construir y administrar un sitio web mediante WordPress.

## ¿Qué habilidad mejoró más?

La habilidad que más mejoró fue la organización y documentación de proyectos web. El alumno aprendió a ordenar carpetas, separar el código fuente, agregar capturas, crear archivos README y subir evidencias a GitHub. También mejoró en el uso de herramientas como PHP, MySQL, WordPress, Git y GitHub.

## ¿Qué aprendiste sobre la asignatura?

El alumno aprendió que el desarrollo de aplicaciones web no solo consiste en escribir código, sino también en organizar correctamente los archivos, probar el funcionamiento de cada proyecto, corregir errores y documentar cada evidencia. También comprendió la importancia de utilizar repositorios en GitHub para presentar el trabajo de forma clara y ordenada.

# Conclusión

Con la elaboración de este portafolio, el alumno integró los proyectos desarrollados durante el semestre y reforzó sus conocimientos sobre desarrollo de aplicaciones web. Los proyectos 02 y 04 fueron los que presentaron mayor dificultad, ya que implicaron trabajar con base de datos, organización de archivos, personalización y pruebas de funcionamiento. Los demás proyectos fueron más sencillos, pero también ayudaron a fortalecer la práctica, la documentación y el uso de herramientas importantes para la asignatura.
